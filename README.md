# EEG Spectral Analysis of PTSD vs Healthy Controls

Reanalysis of the open dataset from **Rahmani et al. (2018, PLoS ONE)**  
[https://doi.org/10.1371/journal.pone.0199144](https://doi.org/10.1371/journal.pone.0199144)

This project investigates alpha (8–12 Hz) and beta (13–30 Hz) EEG power 
differences between PTSD and healthy control participants using MNE-Python.
